Overview

Barstools

This project aims to build and finish barstools for my kitchen.

Project Plan

M1: Barstools are deconstructed


Completion Date: 0-0-0 (Estimate)

In the first milestone, the barstools need to be deconstructed and have all of the hardware removed. This will make it easier to sand and finish. We also want to improve the build quality by using bolts instead.

Dependencies

  1. Order new hardware
  2. Hardware removal

M2: Sanding


Completion Date: 2023-2-27 (Estimate)

In the second milestone, the chairs will be sanded and ready for finishing.

Dependencies

  1. Sanding

M3: Finishing


Completion Date: 2023-3-13 (Estimate)

In the third milestone, we will spray the barstools with a lacquer. It is at this point they will be production-ready.

Dependencies

  1. Lacquer
